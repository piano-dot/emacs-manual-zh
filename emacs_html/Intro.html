Introduction ¶

您正在阅读 GNU Emacs，它是 GNU 对于 Emacs ，一个高级的、自我描述的、可定制的、可扩展的编辑器 Emacs 的呈现。 ( GNU (GNU’s Not Unix) 中的‘G’ 不是静音的。)

我们称 Emacs 为高级的，因为它可以做的不仅仅是简单的插入和删除文本。 它可以控制子进程、自动缩进程序、一次显示多个文件、像编辑本地文件一样编辑远程文件等等。 Emacs 编辑命令以字符、单词、行、句子、段落和页面以及各种编程语言的表达式和注释进行操作。

自我描述 意味着您可以随时使用称为 帮助命令 的特殊命令来查找您的选项，或查找任何命令的作用，或查找与给定主题相关的所有命令。 请参阅See Help。

可定制的 意味着您可以轻松更改 Emacs 命令的简单方法。 例如，如果您使用编程 在以 ‘<**’ 开头并以 ‘**>’ 结尾的注释语言中，你可以告诉 Emacs 注释操作命令 使用这些字符串 （see Manipulating Comments）。 再举一个例子，你 可以重新绑定基本的光标运动命令（上、下、左、右） 到键盘上您觉得舒服的任何键。 See Customization。
