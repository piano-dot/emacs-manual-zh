序言 ¶

本书记录了Emacs的简单使用和个性化方式. 简单的个性化不需要你是一个程序员, 但是如果你不喜欢个性化，请忽略个性化提示.

本书主要是一个参考手册，但也可以作为一个入门手册. 如果您是一个 Emacs 新手，您可以在读本文之外, 从内置的, 边学边做的教程开始学习. 需要运行, 打开 Emacs 然后输入C-h t (就是按住 control 键再按 t 键). 这个教程记录了命令,告诉你什么时候尝试, 同时告诉你结果. 这个教程可以使用多种语言阅读.

初读时，只需浏览第 1 章和第 2 章，它们描述了手册的符号约定和 Emacs 显示屏的一般外观。 请注意这些章节中回答了哪些问题，以便您稍后可以参考。 阅读第 4 章后，您应该练习那里显示的命令。 接下来的几章将介绍经常使用的基本技术和概念。您需要彻底理解它们，因此请尝试它们，直到您流利为止。

第 14 章到第 19 章介绍了对多种编辑有用的中阶功能。 第 20 章和后续章节介绍了可选但有用的功能;在需要时阅读这些章节。

如果 Emacs 似乎无法正常工作，请阅读常见问题一章。(see Dealing with Emacs Trouble), 它解释了如何应对几个常见问题,以及 何时以及如何报告 Emacs 错误 （see Reporting Bugs）。

若要查找特定命令的文档，请查看索引。键（字符命令）和命令名称具有单独的索引。还有一个术语表，每个术语都有一个交叉引用。

本手册可作为印刷书籍和 Info 文件提供。 Info 文件用于从 Emacs 本身读取，或者使用 Info 程序读取。 Info 是 GNU 系统中文档的主要格式。 Info文件和印刷书籍包含基本相同的文本 并且从相同的源文件生成，这些源文件也是 与 GNU Emacs 一起分发。

GNU Emacs 是 Emacs 编辑器家族的一员。 有许多 Emacs 编辑器，它们都共享着共同的组织原则。 有关 Emacs 的基本理念以及从其开发中吸取的经验教训的信息，请参阅Emacs, the Extensible, Customizable Self-Documenting Display Editor(英语)，可以从 https://dspace.mit.edu/handle/1721.1/5736查阅. 本版本的手册主要用于安装在 GNU 和 Unix 系统上的 GNU Emacs。 GNU Emacs 也可以在 MS-DOS、Microsoft Windows 和 Macintosh 系统上使用。 本手册的 Info 文件版本包含有关在这些系统上使用 Emacs 的更多信息。 这些系统使用不同的文件名语法;此外，MS-DOS 不支持所有 GNU Emacs 功能。 请参阅 See Emacs and Microsoft Windows/MS-DOS来查阅在Windows上使用Emacs的信息。 也请参阅See Emacs and macOS / GNUstep来认识在Macintosh和GNUStep上使用Emacs的信息。

分发的条件 ¶

GNU Emacs 是 自由软件;这意味着每个人都可以自由使用它，并且可以在特定条件下自由地重新分发它。 GNU Emacs 不属于公有领域;它是受版权保护的，并且对其分发有限制，但这些限制旨在允许一个好的合作公民想要做的一切。不允许的是试图阻止其他人进一步分享他们可能从您那里获得的任何版本的 GNU Emacs。 确切的条件可以在 Emacs 附带的 “GNU 通用公共许可证”( GNU GENERAL PUBLIC LICENSE) 中找到，也出现在本手册中的 See GNU GENERAL PUBLIC LICENSE.

获得 GNU Emacs 副本的一种方法是从拥有它的人那里。 您无需征得我们的许可，也无需告知其他任何人; 只需复制它。 如果您可以访问 Internet，则可以通过匿名FTP获得 GNU Emacs的最新发行版本;看 https://www.gnu.org/software/emacs 在我们的网站上了解更多信息 信息。

当您购买计算机时，您也可能收到 GNU Emacs。 计算机制造商可以按照适用于以下条件的相同条款自由分发副本： 别人。 这些条款要求他们为您提供完整的源代码，包括他们可能所做的任何更改，并允许您根据通用公共许可证的通常条款重新分发从他们那里收到的 GNU Emacs。 换句话说，当您获得该程序时，该程序必须对您免费，而不仅仅是对制造商免费。

如果您觉得 GNU Emacs 有用，请捐款给自由软件基金会来支持我们的工作。 在美国对自由软件基金会的捐款免税。 如果您在工作场所使用 GNU Emacs，也请建议公司捐款。 要捐款，请参阅 https://my.fsf.org/donate/。 有关您可以提供帮助的其他方式，请参阅 https://www.gnu.org/help/help.html。
