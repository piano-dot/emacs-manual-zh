Emacs 编辑器 ¶

Emacs 是高级, 可扩展, 可自定义, 自述性编辑器.本书介绍如何使用 Emacs进行编辑, 以及一些个性化方式; 本书对应 GNU Emacs 29.2.

For information on extending Emacs, see Emacs Lisp in The Emacs Lisp Reference Manual.
