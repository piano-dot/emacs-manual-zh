# emacs-manual-zh

#### 介绍
一个emacs手册的不完整翻译。

暂时没有完全翻译完，但是欢迎对任意部分提交修改PR。
不变章节不会（在Richard Stallman同意之前）被删除。

编译pdf文件需要你准备一份完整的texinfo，xetex（包含在texlive），查看2024年2月Ralph Lin与Gavin Smith 在gnu的 help-texinfo 邮件列表里的聊天记录。（现在还不能用texi2any,只能在xetex基础上用texi2dvi --pdf）

#### 参与贡献

1.  Fork 本仓库
2.  提交更改
3.  提交Pull Request
